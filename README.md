# Css2Stylus

Ruby gem that converts CSS to Stylus.

## Usage

Install the gem:
```bash
$ gem install css2stylus
```

Convert any css file:
```bash
$ css2stylus myfile.css
```

The output will be saved to `myfile.styl`.
